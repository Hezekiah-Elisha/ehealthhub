### Prerequisites
What things you need to install the software and how to install them

#### Backend
```bash
$ composer create-project laravel/laravel eHealthHub
$ cd eHealthHub

$ php artisan serve
```

#### Frontend
```bash
$ cd ehealthhub
$ npm install
$ npm run serve
```
